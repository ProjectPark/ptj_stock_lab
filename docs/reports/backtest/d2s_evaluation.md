## D2S 전략 개요

D2S(D2S행동추출)는 실거래 행동 패턴을 추출하여 규칙화한 Line C 전략이다.
일봉 기반으로 작동하며 RSI, MACD, Bollinger Band, ATR 등 기술적 지표를 활용한다.
