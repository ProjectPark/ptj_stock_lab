## 2. 매수 규칙 (Buy Rules)

### 2-1. 매수 전 필수 검증

```
[검증 1] GLD 양전? → 비조건부 매수 전체 차단 (조건부 COIN/CONL은 허용)
[검증 2] 종목당 누적 > 1,000만원? → 차단
[검증 3] 물타기 > 7회? → 차단
[검증 4] 같은 종목 5분 이내 주문? → 차단
[검증 5] 당일 이미 매수→매도 완료? → 차단
```

### 2-2. 매수 실행 순서

매수 시그널은 아래 순서로 처리된다. 각 단계는 독립적으로 판단한다.

#### (1) 쌍둥이 ENTRY — 갭 ≥ 1.5%

| 항목 | 값 |
|---|---|
| 조건 | `gap = 선행 등락률 - 후행 등락률 ≥ 1.5%` |
| 매수 대상 | 후행 종목 (MSTU/IRE 중 당일 선택, 또는 CONL) |
| 매수 금액 | 300만원 (초기 진입) |

**코인 후행 선택 (MSTU vs IRE)**:
1. 당일 첫 30분 변동성 계산: `(고가 - 저가) / 시가 × 100`
2. 변동성 차이 ≥ 0.5% → 변동성 큰 종목 선택
3. 변동성 비슷 → 거래량 큰 종목 선택

#### (2) DCA (물타기)

| 항목 | 값 |
|---|---|
| 조건 | 보유 종목이 초기 매입가 대비 -0.5% × n 하락 |
| 금액 | 100만원씩, 최대 7회 |
| 종목당 최대 투입 | 1,000만원 (300 + 100×7) |
| 예외 | BRKU는 DCA 금지 (1회 매수만) |

```
예시:
  초기 진입 $50.00 (300만원)
  DCA 1회: $49.75 (-0.5%) → +100만원
  DCA 2회: $49.50 (-1.0%) → +100만원
  ...
  DCA 7회: $48.25 (-3.5%) → +100만원 (누적 1,000만원)
```

#### (3) CONL 조건부 매수

| 항목 | 값 |
|---|---|
| 트리거 | ETHU, XXRP, SOLT **각각 +3% 이상** |
| 매수 대상 | CONL |
| 매수 금액 | 300만원 |

#### (4) COIN 조건부 매수

| 항목 | 값 |
|---|---|
| 트리거 | ETHU, XXRP, SOLT **각각 +3% 이상** |
| 매수 대상 | COIN |
| 매수 금액 | 300만원 |

#### (5) 하락장 방어 — BRKU

| 항목 | 값 |
|---|---|
| 조건 | Polymarket 코인·나스닥·이더리움 모두 ≤ 20% |
| 매수 대상 | BRKU |
| 매수 금액 | 총 투자금의 10% (200만원) |
| DCA | 금지 (1회만) |

## 3. 매도 규칙 (Sell Rules)

매도는 매수보다 **우선 처리**된다. 아래 순서로 매 1분마다 판단한다.

### 3-1. 매도 우선순위

```
[우선순위 1] 손절 (stop_loss)       → 즉시 전량 매도
[우선순위 2] 시간 손절 (time_stop)   → 5시간 초과 시 매도
[우선순위 3] CONL 매도              → 순수익 +2.8% 또는 트리거 평균 <1%
[우선순위 4] COIN 매도              → 시황별 순수익 기준
[우선순위 5] 분할 매도 (staged)     → 진행 중인 30% 잔여분 처리
[우선순위 6] 쌍둥이 수렴 매도       → 갭 ≤ 0.9% 시 80%+20% 분할
```

### 3-2. 손절 (Stop Loss)

| 시황 | 손절 라인 | 기준 |
|---|---|---|
| 일반/하락장 | **-3.0%** | 전일 종가 대비 등락률 |
| 강세장 (Polymarket ≥ 70%) | **-8.0%** | 전일 종가 대비 등락률 |

### 3-3. 시간 손절 (Time Stop)

| 시황 | 보유 제한 | +2% 순수익 시 |
|---|---|---|
| 일반/하락장 | **5시간** (장중) | 5시간 내 매도 |
| 강세장 | **다음 거래일** | **즉시 매도** (대기 불필요) |

- 5시간 = 정규장 (9:30~16:00) 기준
- 강세장 다음날: **양전**(전일 종가 대비 상승) + **순수익** > 0 → 매도

### 3-4. CONL 매도 (둘 중 하나 충족 시)

| 조건 | 기준 | 행동 |
|---|---|---|
| 수익 실현 | CONL 순수익 ≥ **+2.8%** | 즉시 전량 매도 |
| 트리거 이탈 | ETHU/XXRP/SOLT 평균 < **+1.0%** | 즉시 전량 매도 |

### 3-5. COIN 매도 (시황별)

| 시황 | 조건 | 행동 |
|---|---|---|
| 일반/강세장 | COIN 순수익 ≥ **+3.0%** | 매도 |
| 하락장 | COIN 순수익 ≥ **+0.3%** | 당일 즉시 매도 |

### 3-6. 쌍둥이 수렴 매도 — 0.9% 기준

**매도 조건**: `gap = 선행 등락률 - 후행 등락률 ≤ 0.9%` (후행이 양전일 때)

| 단계 | 대상 | 비율 | 타이밍 |
|---|---|---|---|
| 1차 | 전체 보유 | **80%** | 즉시 |
| 2차 | 나머지의 30% | 전체의 6% | 5분 후 |
| 3차 | 나머지의 30% | 전체의 4.2% | 10분 후 |
| ... | 30%씩 반복 | 점감 | 5분 간격 |

### 3-7. 장 마감 처리 (EOD)

| 시황 | 행동 |
|---|---|
| 강세장 | 포지션을 **carry** (다음 거래일로 이월) |
| 일반/하락장 | 잔여 포지션 **전량 청산** |

## 4. 시황 판단 체계

### 4-1. Polymarket 기반 모드 결정

```
if BTC 상승확률 ≥ 70%:
    → 강세장 (bullish)
elif BTC ≤ 20% AND NDX ≤ 20% AND ETH ≤ 20%:
    → 하락장 (bearish)
else:
    → 일반 (normal)
```

### 4-2. GLD 시그널

| GLD 상태 | 효과 |
|---|---|
| 양전 (+) | 비조건부 매수 전체 금지 (조건부 COIN/CONL은 허용) |
| 음전 (-) | 현금에서 추가 매수 허용 |
| 변동 없음 | 기존 포지션 유지 |
