## 0. 엔진 개요 — taejun_history_2023

### 0-1. D2S(Discretionary-to-Systematic) 변환 개요

이 규칙서는 **실제 거래 행위**에서 통계적으로 추출한 규칙이다. 이론적 설계가 아닌 **행동 복원** 기반.

| 항목 | 값 |
|---|---|
| 분석 기간 | 2025.02.19 ~ 2026.02.12 (248 거래일) |
| 거래 건수 | 953건 (구매 882건, 판매 71건) |
| 종목 수 | 27개 (주력 6개) |
| 총 투자 금액 | $1,590K+ |
| 라운드트립 | 722건 (FIFO 매칭) |
| 승률 | 65.5% (473승 / 249패) |
| 평균 수익률 | +7.39% |

### 0-2. 핵심 용어 정의

> **규칙 충돌 시 우선순위**: R14(GLD↑+SPY↓ 리스크오프) > R1(GLD 억제) > R13(SPY 스트릭)
> 즉, R14 ALL 조건이 충족되면 R1·R13 억제를 무효화하고 적극 매수 모드로 전환한다.

| 용어 | 정의 |
|---|---|
| **매수 억제** | 당일 신규 진입 **전면 금지**. 기존 보유 포지션 관리(이익실현·손절)는 계속 수행. |
| **적극 매수** | 기본 매수 비중을 최대 한도까지 집행. R14 발동 시 적용. |
| **기본 매수** | 표준 비중(15%) 수준 진입. 필터 이상 없을 때의 기준 행동. |
| **소량 매수** | 탐색적 소형 진입. 구체 기준: 매수 금액 ≤ $500(고정) 또는 자본의 5% 이하 중 작은 값. |
| **R14 오버라이드** | GLD↑+SPY↓ ALL 조건 충족 시 R1(GLD 억제)·R3(BTC 관망)·R13(SPY 스트릭 금지) 세 규칙을 일시 무효화하고 적극 매수 모드 진입. |

## 1. 시황 판단 — 다차원 필터

### 1-1. GLD 시황 필터 (R1)

**통계적 유의성**: Welch's t-test p=0.036 (**유의**)

| GLD 구간 | 매수 확률 | vs 기저(44%) | 행동 |
|---|---|---|---|
| GLD < -0.5% | 52% | +8%p | **매수 적극** |
| -0.5% ≤ GLD < 0% | 46% | +2%p | 기본 |
| 0% ≤ GLD < 0.5% | 43% | -1%p | 기본 |
| 0.5% ≤ GLD < 1.0% | 44% | 0%p | 기본 |
| **GLD ≥ +1.0%** | **39%** | **-5%p** | **매수 억제** |

> 근거: 매수일 GLD 평균 +0.01% vs 관망일 +0.45%. GLD 상승은 안전자산 쏠림 = 리스크오프 신호.

### 1-2. BTC 확률 필터 (R3)

Polymarket BTC up 확률을 연속 변수로 활용한다.

| BTC up 확률 | 매수 확률 | 행동 |
|---|---|---|
| **< 0.30** | 낮음 | **매수 억제** (공황 수준 — 단, R14 오버라이드 시 예외) |
| 0.30 ~ 0.40 | 낮음 | 관망 |
| **0.40 ~ 0.50** | **기본** | 기본 매수 |
| 0.50 ~ 0.60 | 기본 | 기본 매수 |
| **0.40 ~ 0.75** (OOS 규칙) | 조건 충족 | **쌍둥이 진입 허용** ← 하한 0.40 적용 |
| > 0.75 | 관망 | **매수 억제** |
| 미수신 | — | **전날 값 계승, 없으면 0.50 (중립) 처리** |

> 근거: 매수일 BTC up 평균 0.48 vs 관망일 0.50 (차이 -0.024). Walk-Forward OOS에서 feature importance 0.172.
> ※ OOS 규칙 하한 0.40 추가: 공황 구간(< 0.30)에서 쌍둥이 진입이 자동 허용되는 오류 방지.

### 1-3. SPY×GLD 2D 매수 확률 매트릭스

SPY와 GLD의 조합으로 시황을 판단한다. **행은 상호 배타적 구간으로 정의한다 (중복 없음).**

| SPY 구간 | GLD 하락 (< 0%) | GLD 상승 (≥ 0%) |
|---|---|---|
| **SPY 강세 (> +1%)** | **최저 매수 (25%)** | 기본 (44%) |
| **SPY 약상승 (0% ~ +1%)** | 기본 (44%) | **최고 매수 (62%)** |
| **SPY 약하락 (0% ~ -1%)** | 기본 (44%) | 기본 (44%) |
| **SPY 약세 (< -1%)** | **역발상 매수 (50%)** | 기본 (44%) |

> 핵심: SPY 약상승 + GLD↑ = 동시 완만 상승 = 62% 최고 매수율. SPY 강세(+1%↑) + GLD↓ = 25% 최저 매수율.
> SPY 약세 + GLD↑ 조합은 R14(GLD↑+SPY↓) 리스크오프 신호로 별도 처리 (1-5절 참조).
> ※ 경계값 처리: SPY = 0.0% 정확히는 "약상승(0~+1%)" 구간에 포함. GLD = 0.0% 정확히는 "GLD 상승(≥0%)" 구간에 포함.

#### R14 오버라이드 규칙 (⚠️ Critical — 타 규칙보다 우선) [Study A 개정]

**GLD↑ + SPY↓** 조건이 충족되고 **SPY 극단 하락이 아닌** 경우, 아래 억제 규칙을 **일시 무효화**한다.

| 구분 | 조건 | 처리 | 근거 |
|---|---|---|---|
| **R14 기본 발동** | GLD_pct > 0% **AND** -1.5% ≤ SPY_pct < 0% | 오버라이드 활성 | 기본 리스크오프 신호 |
| **R14 최적 발동** | GLD_pct ≥ +0.5% **AND** -1.5% ≤ SPY_pct < -0.5% | 오버라이드 + 비중 상향 | Study A Level 2 (p=0.042) |
| **R14 특급 발동** | R14 기본 조건 **AND** 연속 3일+ | 오버라이드 + 최대 비중 | Study A 100% 승률 (n=10) |
| **R14 미발동** | SPY_pct < -1.5% | 오버라이드 없음 + 비중 50% 축소 | Study A 위험 구간 (29.4%) |

| 무효화 대상 | 원래 규칙 | R14 발동 시 처리 |
|---|---|---|
| R1 GLD 억제 | GLD ≥ +1.0% → 매수 억제 | **무효화 → 적극 매수로 전환** |
| R3 BTC 관망 | BTC up 확률 < 0.40 → 관망 | **무효화 → 기본 매수 허용** |
| R13 SPY 스트릭 | SPY 3일+ 연속 하락 → 주의 | **무효화 → 역발상 매수 강화** |

> R14 최종 발동 기준: GLD_pct > 0% **AND** -1.5% ≤ SPY_pct < 0% (당일 종가 기준).
> SPY_pct < -1.5%: R14 미발동 + 당일 신규 진입 비중 50% 축소 (공황 방어).
> 연속 3일+ R14 발동: 최대 비중 허용.

### 1-8. 통합 시황 점수 market_score (Study C 신규)

개별 시황 신호(R1·R6·R13·R14·R16)를 **하나의 연속 점수(0~1)**로 통합한다. 높을수록 매수 적합.

#### 점수 산출 공식

| 신호 | 산출 방법 | 가중치 | 근거 |
|---|---|---|---|
| `gld_score` | GLD_pct: -0.5%→1.0, +1.0%→0.0 정규화 | 0.20 | R1 (p=0.036) |
| `spy_score` | SPY_pct: -1.0%→1.0, +1.0%→0.0 정규화 | 0.15 | R6 (+5.6%p) |
| `riskoff_score` | GLD↑+SPY↓ 강도 × Study A 경계 적용 | 0.25 | R14 (최강) |
| `streak_score` | SPY 3일+ 연속 상승 → 0.0, 기본 → 1.0 | 0.15 | R13 (27.3%) |
| `vol_score` | SPY 5일 변동성 → 고변동=1.0 | 0.15 | R16 (★ 유일 유의) |
| `btc_score` | BITU_pct: -5%→1.0, +5%→0.0 역발상 | 0.10 | R3 대용 |

> `market_score = Σ(가중치 × 신호점수)` → 0~1 클립
> SPY -1.5% 미만 → riskoff_score 강제 0 (Study A 위험 구간)
> 연속 3일+ 리스크오프 → +0.10 보정 (Study A 특급 신호)
