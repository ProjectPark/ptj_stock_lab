```

  Day 1: +2% 회복? → V-바운스 성공 가능, 유지
  Day 2: +2% 회복? → 유지
  Day 3: +2% 회복? → 유지
  Day 4 (진입 후 3일): 미회복 → 전량 손절
```

> **근거**: BB 하단 돌파 후 빠른 회복(≤3일) = 평균 +$590, 느린 회복(8일 강제청산) = 평균 -$805.
