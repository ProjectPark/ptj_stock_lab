## 6. 분석

### 1. 실제 수수료는 표준의 1/3.5 — 유리한 조건

- 거래내역서 기준 실효 수수료 0.10% (표준 0.25%의 40%)
- 환전 스프레드가 별도 부과되지 않아 실질 왕복 0.20% (표준 0.70%의 29%)
- 소액 거래 면제(72건)도 비용 절감에 기여
- **현재 수수료 조건은 단기 매매에 유리**

### 2. 수수료 적용 후에도 전략 판단은 불변

- 갭 시뮬레이션: 3개 페어 모두 마이너스 → **수수료 무관하게 전략 자체가 비수익적**
- 실제 매수 성과: 수수료 후에도 +1.16% 양수 → **매수 타이밍 판단은 유효**
- 5시간 규칙: 수수료 영향 없음 → **규칙 유효성 유지**
- 종목별 수익/손실 순위: 수수료 적용 전후 동일 → **종목 선별 기준 변경 불필요**

### 3. 표준 수수료(0.25%+FX) 적용 시 리스크

- 갭 시뮬레이션 누적 손실 8.2배 확대 (-186% → -1,534%)
- 수렴 청산 평균 +1.09% → +0.38%로 수익성 대폭 감소
- 실제 매수 건 수익 52% 감소 (+1.36% → +0.65%)
- **프로모션 종료 시 또는 타 증권사 이용 시 전략 재검토 필요**

### 4. 수수료 민감도: 손절이 가장 큰 영향

- 손절 시 손실: -3.00% → -3.19%(실제) / -3.68%(표준)
- 수수료가 손절 손실에 추가되므로, 손절 빈도가 높을수록 수수료 부담 가중
- **손절 빈도 감소 = 수수료 절감의 핵심**
