## 권고 사항 (우선순위 순)

1. **[Critical] L1-02 — M200 조건7 행동 기준 통일**: VNQ 5%+ 발동 시 "즉시 매도"(전액)인지 "30% 비중 축소"인지 단일 행동으로 명시. 원문에서도 혼재되어 있으므로 태준님에게 최종 확인 필요.

2. **[Critical] L2-01 — M200 해제/복귀 조건 추가**: 7개 조건 각각에 대해 발동 후 정상 운영 복귀 시점 정의. 특히 조건5(20일선 이탈)는 지속 상태이므로 복귀 기준 없으면 영구 정지 위험.

3. **[Critical] L3-02 — L전략 Polymarket 수치 재확인**: 원문(0222 line 1133)은 "20% 이하"이고 정리 문서(line 599)는 "30% 이하". ✅ 해결 표시 전 실제 원문 수치와 다른 값이 반영됨. 태준님 재확인 후 정정 필요.

4. **[Critical] L4-01 — 재난 이머전시와 M200 충돌 처리**: Polymarket 급변으로 재난 이머전시 진입 시 동시에 M200 조건이 달성될 경우 우선순위 규칙 명시. M200이 항상 우선인지, 재난 이머전시 상태에서는 M200 유예인지 결정 필요.

5. **[Critical] L4-02 — M80과 J전략 중복 종목 처리**: CONL, ROBN, MSTU 등 동일 종목이 M80과 J 양쪽에서 매수 대상으로 존재. P-7 해결 전까지 중복 매수 방지 임시 규칙 명시 권고.

6. **[Critical] L6-01 — M80 진행 중 M200 발동 시 미체결 주문 처리**: M80 T2/T3 진입을 위한 지정가 주문이 대기 중일 때 M200 발동 시 주문 취소 여부를 명시.

7. **[Critical] L6-02 — M80 다섹터 동시 발동 자본 배분**: 여러 섹터 동시 조건 달성 시 M5 T1~T4 체계가 각 섹터에 독립 적용되면 자본 초과 발생. 섹터 간 자본 배분 우선순위 또는 합산 한도 명시 필요.

8. **[Warning] L1-03 — M7 비고 선행지수 목록 정정**: 비고란 "선행지수는 변경하지 않음: BITU, ROBN, NVDL"에서 ROBN, NVDL은 이미 교체 대상으로 확정됨. BITU만 선행지수 유지 대상으로 정정 필요.

9. **[Warning] L5-06 — M200 조건5 적용 범위 통일**: M200 섹션(line 229)에서는 "모든 주식", M80 섹션 재현(line 923)에서는 "해당 종목"으로 상이. 단일 기준으로 통일.

10. **[Warning] L6-04 — 전략 B(잽모드 BITU)와 M7 BITU 매매금지 연동**: M7에서 BITU를 매매금지로 정정했으나 전략 B가 삭제되지 않음. 전략 B 운영 상태(정지/폐기/유지) 명시 필요.
