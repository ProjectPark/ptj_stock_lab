## 1. 시황 판단 (매매 전 필수 확인)

매매 전 아래 순서로 시장 상태를 확인한다. **4가지 시장 모드** 중 하나로 판정된다.

| 순서 | 항목 | 판단 기준 | 결과 |
|---|---|---|---|
| 1 | **Polymarket (강세)** | 상승확률 70% 이상 | 70% 이상 → **강세장 모드** |
| 2 | **Polymarket (하락)** | 코인·나스닥·이더리움 상승확률 모두 20% 이하 | 3조건 충족 → **하락장 모드** |
| 3 | **횡보장 감지** | 아래 5개 지표 중 3개 이상 충족 | **횡보장 모드** → 현금 100% |
| 4 | **금(GLD)** | GLD 등락률 확인 | 양전 → 매매 금지 권고 |

### 1-1. 시장 모드 분류

```
Polymarket ≥ 70%           → 강세장 모드
Polymarket 3종 ≤ 20%       → 하락장 모드
횡보장 지표 3/5 이상 충족    → 횡보장 모드  [v3 신규]
위 해당 없음                → 일반 모드
```

- **강세장**: 손절 -3% → -8% 완화, 보유 다음날까지 연장.
- **하락장**: BRKU 10% + 현금 90%.
- **횡보장**: **현금 100%. 모든 신규 매수 금지.** 기존 보유분은 손절·매도 규칙에 따라 청산만 진행.
- **일반**: 기본 규칙대로 매매.

> 모드 우선순위: **횡보장 > 하락장 > 강세장 > 일반**. 횡보장 조건이 충족되면 다른 모드보다 우선 적용한다.

### 1-2. 횡보장 감지 — 5개 지표 (v3 신규)

아래 **5개 지표 중 3개 이상** 충족 시 횡보장으로 판정한다.

| # | 지표 | 횡보장 조건 | 근거 |
|---|---|---|---|
| 1 | **Polymarket 확률** | 상승확률 **40~60%** 범위 | 방향성 없음 = 횡보 |
| 2 | **GLD 등락률** | \|GLD 등락률\| ≤ **0.3%** | 금 방향 애매 = 시장 비결정 |
| 3 | **쌍둥이 갭 패턴** | 당일 갭 ≥ 2.2% 시그널 발생 후 **수렴 실패** 2회 이상 | 갭 벌어지나 수익 안 남음 |
| 4 | **COIN/CONL 트리거** | 트리거 발동 후 **목표 수익 미달** 2회 이상 | 모멘텀 시그널 불발 |
| 5 | **SPY·QQQ 등락률** | SPY와 QQQ 모두 \|등락률\| ≤ **0.5%** | 지수 자체가 방향 없음 |

**판정 로직**:
- 5개 지표를 매 **30분** 간격으로 평가한다.
- **3개 이상 충족** → 횡보장 모드 진입 → **모든 신규 매수 즉시 중단**.
- **2개 이하 충족** → 횡보장 해제 → 해당 시점의 다른 모드(강세/하락/일반)로 복귀.
- 횡보장 진입·해제 시 **로그 기록** (시각, 충족 지표, 판정 결과).

### 1-3. 횡보장 모드 행동 규칙

| 행동 | 설명 |
|---|---|
| **신규 매수** | **전면 금지** (쌍둥이 ENTRY, 조건부 매매, 물타기 모두 차단) |
| **기존 보유 매도** | 정상 작동 (0.9% 매도, 손절, 시간 손절 모두 유효) |
| **현금 비중** | **100%** (기존 보유분 청산 완료 시) |
| **BRKU** | 매수하지 않음 (하락장이 아닌 횡보장이므로) |
| **모드 해제** | 30분 재평가에서 지표 2개 이하 충족 시 자동 해제 |

> **핵심 원칙**: 횡보장에서는 **아무것도 안 하는 것이 최선**이다. 방향이 없는 시장에서 매매하면 수수료만 쌓이고 수익은 나지 않는다.

### 1-4. 기존 모드 규칙

- GLD가 양전(+)이면 **모든 매수를 금지**한다 (v1 동일).
- GLD가 음전(-)이면 **조건부 매수 허용** — 나머지 현금으로 추가 매수 가능.
- Polymarket 상승확률 **70% 이상** → **강세장 모드** 진입:
  - 손절 라인 -3% → **-8%** 완화
  - 보유 시간 5시간 → **다음 거래일까지** 연장 (다음날 양전 + 순수익 시 매도)

## 3. 쌍둥이 매매

### 3-1. 페어 구성

| 페어 | 선행(lead) | 후행(follow) |
|---|---|---|
| 코인 | BITU | MSTU, **IRE** |
| 은행 | ROBN | CONL |

### 3-2. 코인 후행 종목 선택 — MSTU vs IRE (당일 1종목만)

코인 페어의 후행 종목은 MSTU, IRE 중 **당일 1개만 선택**하여 거래한다.

**선택 기준** (순서대로 적용):

| 순서 | 기준 | 선택 |
|---|---|---|
| 1 | 변동성 차이 ≥ 0.5% | **변동성이 큰 종목** 선택 |
| 2 | 변동성 비슷 (차이 < 0.5%) | 거래량·선호도·기대치 비교 |

- 당일 선택된 종목만 매수·매도 대상이 된다. 나머지는 거래하지 않는다.

### 3-3. 매수 규칙 (ENTRY) — v3 강화

**갭 계산**: `gap = lead 등락률 - follow 등락률`

| 조건 | 시그널 | 행동 |
|---|---|---|
| gap ≥ **2.2%** | ENTRY | 갭 벌어짐 → 후행 종목 매수 |
| gap < 2.2% | HOLD | 관망 |

> **v3 변경**: 1.5% → **2.2%**. 애매한 소폭 갭을 무시하고 확실한 기회에만 진입한다. 이것이 거래 횟수 감소에 가장 큰 효과를 준다.

### 3-4. 매도 규칙 — 0.9% 매도

후행 종목이 선행에 **따라붙었을 때** 매도한다. 지정가(LIMIT) 주문으로 실행한다.

**매도 조건** (둘 중 하나 충족 시):

| 조건 | 설명 | 예시 |
|---|---|---|
| 후행 ≥ 선행 | 후행이 선행을 따라잡거나 초과 | 선행 +3%, 후행 +3.1% |
| 후행 ≥ 선행 - 0.9% | 후행이 선행 대비 0.9% 이내 도달 | 선행 +3%, 후행 +2.1% |

**매도 수량**:

| 단계 | 대상 | 비율 | 타이밍 |
|---|---|---|---|
| 1차 매도 | 전체 보유 | **80%** | 0.9% 매도 조건 충족 즉시 |
| 2차 매도 | 나머지(20%)의 30% | 전체의 **6%** | 1차 후 5분 |
| 3차 매도 | 나머지의 30% | 전체의 **4.2%** | 2차 후 5분 |
| ... | 잔여분의 30%씩 | 계속 감소 | 5분 간격 반복 |

### 3-5. 매도 우선순위

- 코인 후행은 3-2에서 **당일 선택된 1종목만** 매도 대상이다.
- 0.9% 매도 조건 충족 시 해당 종목을 매도한다.

## 5. 손절 규칙

### 5-1. 기본 손절 (일반 시황)

| 조건 | 행동 |
|---|---|
| 종목 등락률 ≤ **-3.0%** | 즉시 매도 |

### 5-2. 완화 손절 (강세장 모드 — Polymarket 상승확률 70% 이상)

| 항목 | 일반 모드 | 강세장 모드 (Polymarket ≥ 70%) |
|---|---|---|
| 가격 손절 | **-3%** | **-8%** |
| 시간 제한 | **5시간** | **다음날까지 연장** |
| 다음날 매도 | - | 양전 + **순수익**(수수료 제외) 발생 시 매도 |
| 수익 +2% 이상 시 | 5시간 내 매도 | **즉시 매도** (다음날 대기 안 함) |

- Polymarket 확률이 70% 미만으로 떨어지면 즉시 일반 모드(-3%, 5시간)로 복귀.

### 5-3. 시간 기반 손절

| 시황 | 보유 제한 | 순수익 +2% 이상 시 | 행동 |
|---|---|---|---|
| 일반 | 매수 후 **5시간** | 5시간 내 매도 | 수익/손실 무관 자동 매도 |
| 강세장 (Polymarket ≥ 70%) | **다음 거래일** | **즉시 매도** | 다음날 양전 + 순수익 시 매도 |

- **5시간 기준**: 매수 시점부터 **장중 거래시간** 기준으로 계산한다.
- **장중** = **프리마켓 + 정규장**. 애프터마켓은 포함하지 않는다.
- 가격 손절과 시간 손절 중 **먼저 도달하는 조건**으로 매도한다.

## 7. 매수 금액 기준

### 7-1. 총 투자금

| 항목 | 금액 |
|---|---|
| **총 투자금** | **2,000만원** |
| 중복 주문 방지 간격 | **20분** (같은 종목+방향 연속 주문 차단) |

> **v3 변경**: 5분 → **20분**. 시장 흔들림에 의한 초단타 연속 매매를 방지한다.

### 7-2. 분할 매수 전략 (종목당) — v3 축소

1개 종목에 대한 매수는 **초기 진입 + 하락 시 물타기** 방식으로 진행한다.

| 단계 | 조건 | 매수 금액 | 누적 투입 |
|---|---|---|---|
| 1차 (초기 진입) | 시그널 발생 시 | **300만원** | 300만원 |
| 2차 (물타기) | 1차 대비 -0.5% 하락 | **100만원** | 400만원 |
| 3차 (물타기) | 1차 대비 -1.0% 하락 | **100만원** | 500만원 |
| 4차 (물타기) | 1차 대비 -1.5% 하락 | **100만원** | 600만원 |
| 5차 (물타기) | 1차 대비 -2.0% 하락 | **100만원** | 700만원 |

- 초기 진입: **300만원** (총 투자금의 15%)
- 하락 시 추가 매수: **100만원씩**, 0.5% 간격, **최대 4회**
- 1종목 최대 투입: 300 + (100 x 4) = **700만원** (총 투자금의 35%)
- 나머지 **1,300만원은 현금 보유**.

> **v3 변경**: 물타기 7회(최대 1,000만원) → **4회(최대 700만원)**. 과도한 물타기를 방지하고 손실 폭을 줄인다. 현금 비중이 50% → 65%로 증가하여 방어력 강화.

### 7-3. 현금 운용 (조건부 추가 매수)

나머지 현금 1,300만원은 아래 조건 충족 시에만 추가 매수에 사용한다.

| 조건 | 행동 |
|---|---|
| **금(GLD) 하락**(음전) | 현금으로 추가 매수 허용 |
| **Polymarket 상승확률 ≥ 70%** | 강세장 모드 → 손절 -8% + 보유 다음날까지 + 현금 매수 허용 |
| 위 조건 미충족 | 현금 유지, 추가 매수 금지 |

### 7-4. 자금 배분 요약

```
총 투자금 2,000만원
├── 1종목 분할매수  최대 700만원 (300 + 100x4)   [v3: 1,000→700]
└── 현금 보유       최소 1,300만원                [v3: 1,000→1,300]
    └── 조건 충족 시 → 추가 매수 가능
        ├── GLD 음전 (금 하락)
        └── Polymarket 강세
```

## 8. 진입 시간 제한 — v3 신규

### 8-1. 매수 허용 시간대

**신규 매수(ENTRY)는 장 초반 2시간에만 허용한다.**

| 시간대 (ET) | 매수 | 매도 | 비고 |
|---|---|---|---|
| 프리마켓 08:00~09:30 | **허용** | 허용 | 프리마켓 전체 |
| 정규장 09:30~10:30 | **허용** | 허용 | 정규장 첫 1시간 |
| 정규장 10:30~16:00 | **금지** | 허용 | 매도·손절만 가능 |

- **매수 허용**: 프리마켓 시작(08:00 ET)부터 정규장 개시 후 1시간(10:30 ET)까지 = **약 2.5시간**.
- **10:30 ET 이후**: 신규 매수 금지. 기존 보유 종목의 **매도·손절만 가능**.
- 물타기도 10:30 ET까지만 허용한다.

> **v3 신규**: 장 후반의 의미 없는 매매를 제거한다. 갭 기반 전략은 장 초반에 기회가 집중되므로, 후반 진입은 효율이 낮다.

### 8-2. 종목당 일일 1회 거래 제한 — v3 신규

**같은 종목은 하루에 1트레이드(매수→매도 1세트)만 허용한다.**

| 규칙 | 설명 |
|---|---|
| 매수→매도 완료 | 당일 해당 종목 **거래 종료** |
| 재매수 | **금지** (다음 거래일까지 대기) |
| 다른 종목 | 별도 카운트 (종목별 독립) |

- 1종목 1일 1트레이드 원칙으로 **자동 선별 매매**가 이루어진다.
- 이 규칙은 물타기(동일 진입 건의 추가 매수)에는 적용되지 않는다. 물타기는 동일 트레이드의 연장이다.
- 손절로 매도된 종목도 당일 재매수 금지 대상이다.

## 10. 주문 실행

### 10-1. 실행 모드

| 모드 | 설명 |
|---|---|
| **manual** | 대시보드에서 수동 주문 (기본값) |
| **auto** | 시그널 감지 → 자동 주문 실행 |

- `auto` 모드는 `auto_trade_enabled = True`일 때만 동작한다.
- 킬스위치(`auto_trade_enabled = False`)로 언제든 자동매매를 중단할 수 있다.

### 10-2. 수익 기준 — 수수료 제외 순수익 (절대 원칙)

**본 문서에서 "수익"이란 모든 곳에서 수수료를 제외한 순수익을 의미한다.**

| 적용 대상 | 기준값 | 의미 |
|---|---|---|
| 손절 완화 모드 다음날 매도 | 양전 + 순수익 | 수수료 뺀 후 이익이어야 매도 |
| Take Profit (+2%) | 순수익 +2% | 수수료 차감 후 +2% |
| CONL 수익 실현 (+2.8%) | 순수익 +2.8% | 수수료 차감 후 +2.8% |
| 쌍둥이 갭 수렴 매도 | - | 갭 기준 (수수료 별도 고려) |

### 10-3. 환율 — 환차익 극대화 (절대 원칙)

**고정 환율을 사용하지 않는다. 실시간으로 가장 유리한 환율에 환전한다.**

| 시간대 | 환전 가능 여부 |

| 정규장 외 시간 | **환전 실행** — 가장 유리한 타이밍 분석 후 진행 |

### 10-4. 주문 유형 — 지정가 전용 (절대 원칙)

## 11. 자동매매 루프 (auto 모드) — v3 수정

Redis `ptj:latest`에서 매 `refresh_interval_sec`(1초)마다 시그널을 폴링하여 아래 순서로 판단한다.

```
── 매도 로직 (항상 실행) ──────────────────────────────
1. 손절 체크        → 기본 -3% (강세장 모드 시 -8%) 이하 종목 즉시 매도
2. 시간 손절 체크    → 5시간 초과 보유 종목 매도
3. CONL 매도 체크   → 수익 +2.8% 또는 트리거 평균 <1% → 매도
4. 쌍둥이 SELL      → 갭 ≤ 0.9% → 80% 즉시 매도 + 20% 5분 분할

── 매수 게이트 (순서대로 체크, 하나라도 걸리면 매수 전체 스킵) ──
5. 횡보장 체크       → 횡보장 모드면 매수 로직 전체 스킵             [v3 신규]
6. 시간 필터 체크    → 10:30 ET 이후면 매수 로직 스킵               [v3 신규]
7. 일일 거래 체크    → 당일 1트레이드 완료 종목은 매수 스킵           [v3 신규]

── 매수 로직 (게이트 통과 시에만 실행) ────────────────
8. 쌍둥이 ENTRY     → 갭 ≥ 2.2% → 초기 300만원 진입              [v3: 1.5→2.2%]
9. 물타기 체크       → 보유 종목 -0.5% 하락마다 100만원 추가 (최대 4회) [v3: 7→4회]
10. CONL 조건부 매수 → ETHU·XXRP·SOLT 각각 +4.5% → CONL 매수    [v3: 3→4.5%]
11. 조건부 매수      → GLD 음전 + 시그널 → 현금에서 추가 매수
12. 하락장 방어      → SPY+QQQ 음전 시 방어주 매수 검토

── 횡보장 재평가 (30분 간격) ──────────────────────────
13. 횡보장 지표 평가  → 5개 지표 중 3개 이상 충족 → 횡보장 모드 ON/OFF [v3 신규]
```
