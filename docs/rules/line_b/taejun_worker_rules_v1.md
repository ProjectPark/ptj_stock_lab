## 개요

W1~W33 Worker 전략 룰 문서. 공통 게이트/주문/비중 규칙은 v2(M1~M300)를 그대로 따르고, 본 문서는 **개별 Worker 진입·청산 조건만** 기술한다.

### Worker 전략 패턴 분류

| 패턴 | 설명 | 해당 Worker |
|------|------|------------|
| **패턴 G** (괴리 단타) | 2배 리더 종목 대비 레버리지 대상 종목 수익률 괴리 포착 → 진입, ±0.2% 수렴 시 청산 | W1~W6, W12~W21, W22~W27, W28~W33 |
| **패턴 V** (거래대금 스캘프) | 3년 최저 거래대금 대비 N배 발동 → +1% 익절 / -10% 손절 | W7~W11 |
| **스윙** | 과거 2년 최고가 대비 -80% 진입 → +200% 목표 / -10% 손절 | 전 Worker |
| **VIX 손절** | UVIX +4% 이상 상승 시 즉시 매도 | W1~W8 명시 (W9~W33: AMB-3 확인 필요) |
| **패턴 M** (VMA) | 전일 20일 평균거래량 +70% & 양봉 → 다음 거래일 매수 → +10% 목표 / 10거래일 제한 | W1~W33 전체 |

### 공통 규칙 (패턴 G)

| 항목 | 내용 |
|------|------|
| 진입 조건 | 리더 +3% 시 대상 +1% 이면 매수 |
| 청산 조건 | 리더 수익률 - 대상 수익률 오차 ±0.2% 이내 시 매도 |
| 주문 방식 | 지정가 ONLY (M1 공통) |
| MMM 게이트 | 거래대금 + 금(GLD) + Polymarket 3개 모두 PASS 필수 (M5 공통) |

### 공통 규칙 (패턴 V)

| 항목 | 내용 |
|------|------|
| 익절 | +1.0% |
| 손절 | -10% |
| 거래대금 기준 | 종목별 3년 최저 대비 N배 (아래 각 Worker 참조) |

### 공통 규칙 (스윙)

| 항목 | 내용 |
|------|------|
| 진입 | 과거 2년 내 최고가 대비 -80% 하락 시 매수 |
| 목표 | +200% 수익률 달성 시 매도 |
| 손절 | 매수 후 -10% 하락 시 매도 |

### 공통 규칙 (패턴 M — VMA)

| 항목 | 내용 |
|------|------|
| 진입 조건 | 전일(일봉) 거래량 ≥ 20일 평균거래량 × 1.7(+70%) AND 전일 양봉(종가 > 시가) |
| 매수 시점 | 다음 거래일 |
| 목표 | +10% 익절 |
| 시간 제한 | 10거래일 이내 +10% 미달 시 10거래일째 장마감 30분 전 강제 청산 |
| MMM 게이트 | ⬜ AMB-1 확인 필요 |

## 모호사항 — 태준님 확인 필요

| ID | 내용 | 관련 항목 |
|----|------|----------|
| **AMB-1** | VMA 전략(패턴M) MMM 게이트(거래대금+금+Polymarket) 적용 여부 — 독립 진입인지, 동일 게이트 통과 필수인지 | 패턴M 전체 |
| **AMB-2** | 동일 종목에 VMA + 스윙 조건 동시 성립 시 우선순위 및 동시 포지션 허용 여부 | 패턴M, 스윙 |
| **AMB-3** | VIX 손절(UVIX +4%) 공통 적용 범위 — W1~W8에만 명시. W9~W33도 동일 적용인지 | VIX 손절 |
| **AMB-4** | W28 ERX 리더 구조: CCJ(1배, 우라늄 기업)가 에너지 섹터 리더로 적합한지. 1배 리더→3배 대상 구조가 패턴G 기준(2배→2~3배)과 다름 | W28 |
| **AMB-5** | W12 SOXL: 리더 NVDL(2배) → 대상 SOXL(3배). 타 반도체는 2배→2배인데 W12만 2배→3배 구조 | W12 |
| **AMB-6** | W29 GDXU VMA 전략에서 리더 불명확 (VICI vs VNQ). VMA는 종목 자체에만 적용되는지 확인 필요 | W29, AMB-1 |
