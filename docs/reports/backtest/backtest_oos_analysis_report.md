## 5. SOXL 교체 분석

### 5.1 PTIR → SOXL 교체 이유

PTIR의 높은 변동성(고점 대비 -47.5%)과 레버리지 ETF 구조적 한계(volatility decay)를 고려해, SOXL(반도체 3배 ETF)로 교체 시 OOS 성과를 비교했다.

### 5.2 SOXL OOS 가격 구조

| 항목 | 값 |
|-----|---|
| OOS 시작가 | $47.24 |
| OOS 고점 | $70.47 (2026-01-29) |
| OOS 말가 | $63.29 |
| 고점比 하락 | -10.2% |
| PTIR 고점比 하락 | -47.5% |

### 5.3 SOXL OOS 백테스트 결과 (btc_rsi_e_loose T30/S-12/H20)

| 종목 구성 | IREN PnL | SOXL/PTIR PnL | 합계 |
|---------|---------|-------------|-----|
| IREN + CONL + **SOXL** | +76만원 | -40만원 | **+35만원** |
| IREN + CONL + **PTIR** | +76만원 | 0원 (0거래) | **+76만원** |

**SOXL 거래 실패 원인**:
- SOXL은 1월 초 vol_ratio 낮음 (0.61~1.23, 기준 1.42 미충족)
- 1월 21일에야 신호 발생 ($55.63 진입)
- 진입 시점이 고점($70.47) 8일 전 → 30% 익절 불발 → 손절 -40만원

**→ 결론**: PTIR 원본 유지가 OOS에서 더 유리. PTIR는 0거래(손실 없음), SOXL은 늦은 진입으로 손실.

## 7. 신호 계보 정리

```
v1~v5 기본 신호
    ↓
exp_A~D (개별 신호 탐색)
    ↓
exp_E (22개 조합 × 26 파라미터 = 572 전수 탐색)
  → 챔피언: di_cross_surge_btc2 (DI갭 + surge1 + BTC regime=2)
    ↓
exp_L (하이브리드: di_cross_surge_btc2 + 다중 신호)
    ↓
Optuna 보수적 최적화 (5개 전략별)
  → btc_rsi_e: BTC RSI14≤64.62, DI갭≥7.82, pct_MA20≥12.76%
  → macd_vol: MACD히스토≥-0.024, RSI14≥68.78, BTC RSI≤69.67
    ↓
앙상블 (btc_rsi_e AND macd_vol 동시 충족)
  → 배포: ptj_stock/backend/app/core/signals.py
```
