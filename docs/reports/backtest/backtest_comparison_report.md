## 6. 핵심 발견

### v1/v2 공통

1. **수수료가 전략을 죽인다**: v1은 수수료 적용 시 +4.12% → -39.75%. v2는 수수료만 1,100만원 (초기 자금의 55%)
2. **손절(-3%)이 핵심 손실원**: v1에서 120회(-$602), v2에서 125회(-834만원). 레버리지 2x 종목 특성상 -3% 빈번
3. **CONL이 문제 종목**: v1에서 27회 손절(-$173), v2에서 CONL 전체 P&L -442만원
4. **조건부 COIN 매매가 가장 안정적**: v1에서 유일한 수익원(+$47, 61% 승률)
5. **시장 환경 의존**: v1 Q2-Q3에만 수익, 나머지 손실. 강세장에서만 유효

### v2 파라미터 최적화 결론

| 결론 | 상세 |
|---|---|
| **즉시 적용 가능** | `CONL_TRIGGER_PCT = 5.0`, `COIN_TRIGGER_PCT = 5.0` (+4.4pp 개선) |
| **손절 완화 금지** | -5%로 완화하면 -43.71%로 대폭 악화 (건당 손실 증가) |
| **분할매도 유지** | 수수료 절감 효과보다 가격 평균화 효과가 더 큼 |
| **DCA 유지** | 7회 → 3회 제한 시 -1.09pp 악화. 물타기 효과 존재 |
| **복합 최적화 실패** | T3(손절 완화)의 악영향이 다른 개선을 압도 |

## 7. 전략 개선 제안

### 단기 (즉시 적용 가능)

1. **CONL/COIN 트리거 +3% → +5% 상향**: 수익률 4.4pp 개선, 수수료 146만원 절감
2. **CONL 매매 자체를 재검토**: v1/v2 모두에서 최대 손실 종목

### 중기 (구조 변경)

3. **거래 빈도 축소**: 수수료가 전략의 근본 문제. 진입 기준을 전반적으로 상향
4. **분할매도 비율 조정**: 현행 80%/30% 유지하되 간격 확대 검토
5. **시간 손절 단축**: 4시간+ 보유에서만 v1이 수익이었으므로, 단기 손절 빈도를 줄이는 방향

### 장기 (전략 재설계)

6. **수수료 최소화 중심 설계**: 국내 증권사 수수료(편도 ~0.35%) 하에서는 일중 고빈도 매매가 구조적으로 불리
7. **레버리지 비율 축소**: 2x ETF의 일중 변동성(-3% 손절 빈번)이 핵심 손실원
8. **Polymarket 활용 강화**: bullish 모드에서만 매매, 나머지는 현금 보유 비중 확대
