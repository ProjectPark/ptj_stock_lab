## 개요

W1~W33 Worker 전략 룰 문서. 공통 게이트/주문/비중 규칙은 v2(M1~M300)를 그대로 따르고, 본 문서는 **개별 Worker 진입·청산 조건만** 기술한다.

### RuleKey 분리 원칙

- 조건이 유사하더라도 각 Worker의 성과(승률/수익률/트리거 빈도)를 개별로 확인하기 위해 Worker는 **RuleKey(룰 ID) 기준으로 분리**하여 기록·집계한다.
- 평가 로직은 if/elif가 아닌 **다중 충족 허용**: 동일 시점에 여러 조건이 충족되면 충족된 RuleKey를 모두 기록하고, 결과 값은 rule_key 기준으로 각각 도출한다.
- **동일 티커는 하루 1회만 신규 진입 허용 (중복 매수 금지)**
- 상위 패턴 체결 시 동일 티커 하위 패턴은 자동 차단 (실행 우선순위표 기준)

### 실행 우선순위

```
VIX손절 > 손절/청산 > SE > SP500 > 스윙 > 단타(G/V) > 패턴M(최근20일)
```

> ※ SE·SP500 vs 패턴G/V 동시 발동 처리: ⬜ P-W-05 확인 필요

### Worker 전략 패턴 분류

| 패턴 | 설명 | 해당 Worker |
|------|------|------------|
| **패턴 G** (괴리 단타) | 리더 종목 대비 대상 종목 수익률 괴리 포착 → 진입, ±0.2%p 수렴 시 청산 | W1~W6, W12~W21, W22~W27, W28~W33 |
| **패턴 V** (거래량 스캘프) | 3년 최저 거래량 대비 N배 발동 → +1% 익절 / -10% 손절 | W7~W11 |
| **스윙** | 과거 2년 최고가 대비 -80% 진입 → +200% 목표 / -10% 손절 | 전 Worker |
| **VIX 손절** | UVIX +4% 이상 상승 시 즉시 매도 | W1~W9 명시 (W10~W33: ⬜ P-W-04) |
| **패턴 M** (VMA) | 전일 20일 평균거래량 +70% & 양봉 → 다음 거래일 매수 → +10% 목표 / 10거래일 제한 | W1~W33 전체 |
| **패턴 S** (숏 거래량) | 전일 20일 평균거래량 +70% & 음봉 → 다음 거래일 인버스 ETF 매수 | SW1, SW4 |
| **SSWW** (공매도 수량) | 공매도 가능 수량 1달 최저 대비 +26.3% 상승 시 매수 중단·보유 매도 | W1~W33 전체 |
| **SE** (반감기 이벤트) | 비트코인 반감기(H0) 당일 매수 → 6개월 또는 12개월 보유 | SE1~SE16 |
| **SP500** (편입 이벤트) | SP500 신규 편입 기업 → 다음 거래일 매수 | SP500 이벤트 |

### 공통 규칙 (패턴 G)

| 항목 | 내용 |
|------|------|
| 진입 조건 | 리더 +3% 시 대상 +1% 이하이면 매수 (스프레드 ≥ 2.0%p) |
| 청산 조건 | \|리더 수익률 − 대상 수익률\| ≤ 0.2%p 시 매도 |
| 장 마감 미청산 | ⬜ **RV-A01**: 이월 허용 여부 및 수익률 기준 리셋 정책 확인 필요 |
| 주문 방식 | 지정가 ONLY (M1 공통) |
| MMM 게이트 | 거래량 + 금(GLD) + Polymarket 3개 모두 PASS 필수 (M5 공통) |
| 비고 | ⬜ **RV-01**: W28·W30~W33의 리더는 1배 종목(CCJ/CVX). 2배 리더 전제와 불일치 — P-W-03 참조 |

### 공통 규칙 (패턴 V)

| 항목 | 내용 |
|------|------|
| 익절 | +1.0% |
| 손절 | -10% |
| 진입 시점 | ⬜ **RV-02**: W10만 "다음 거래일 지정가 매수" 명시 — W7·W8·W9·W11 진입 시점 미정의. P-W-01 참조 |
| 거래량 기준 | 종목별 3년 최저 거래량 대비 N배 (아래 각 Worker 참조) |

### 공통 규칙 (스윙)

| 항목 | 내용 |
|------|------|

|------|------|

### 공통 규칙 (SSWW — 공매도 수량 모니터링)

| 항목 | 내용 |
|------|------|
| 데이터 소스 | iborrowdesk.com/report/{기초 티커} |
| 트리거 | 과거 1달 기준 최저 공매도 가능 수량 대비 당일 가능 수량 +26.3% 이상 상승 |
| 수치 근거 | ⬜ **RV-A05**: 26.3%의 산출 근거 및 "1달"의 영업일/역월 기준 불명 |
| 매수 중단 | 트리거 발동 시 해당 Worker 신규 진입 중단 |
| 청산 대상 | ⬜ **RV-A12**: "해당 종목 전량 매도"의 범위 — 해당 Worker 포지션만 vs 시스템 전체 보유분 (P-W-06 연계) |
| 시간제한 | 트리거 후 1달 (재진입 금지) |
| 시간제한 해제 | 트리거 후 1달 내 최고가 대비 현재가 -30% 이하 하락 시 해제 |
| 해제 후 재진입 | ⬜ **RV-A03**: 재진입 허용 여부 및 신규 SSWW 모니터링 기준 시점 미정의 |
| 패턴M 관계 | ⬜ **RV-A10**: SSWW 청산 후 재진입 금지 기간 중 패턴M 진입 허용 여부 미정의 |
| 데이터 결측 | ⬜ iborrowdesk 장애/미업데이트 시 트리거 판단 보류 + 알람 처리 필요 |
